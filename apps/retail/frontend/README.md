# InterLogIQ

A comprehensive Fleet Maintenance Management System built with React, TypeScript, and PrimeReact.

## Features

- Fleet Management Dashboard
- Vehicle Maintenance Tracking
- Service History Management
- Driver Management
- Maintenance Scheduling
- Reporting and Analytics

## Getting Started

```bash
npm install
npm run dev
```

## Technologies

- React 18
- TypeScript
- PrimeReact
- Vite
- PrimeFlex
